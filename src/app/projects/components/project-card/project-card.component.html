Minesweeper
Java - JavaFX

My very first graphical project using Java. I used the JavaFX library to create a Minesweeper game that works in the same way and with the same rules as the classic Minesweeper. This program can be installed on any machine running Windows with an x64 processor, or MacOS with an M-series (ARM) processor, without any prerequisites thanks to the following installers.
